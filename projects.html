PROJECTS

Minesweeper

I recreated this classic game using Java along with its AWT and Swing packages. The clone has all the functionality of the original game including flagging bombs, choosing difficulty of play, uncovering cells, and detonating bombs.

To-Do List Chrome Extension

Utilizing the Chrome API and javascript, I create an application that allows its users to easily organize a list of tasks. Tasks are saved within the chrome browser and can be re-accessed even after the browser session is closed.

Workout Generator and Log

This command-line based program generates exercise sets based on a users desired workout duration. In addition, users can 'save' a workout within and track their exercise history in a workout log. This program was written using Python.
